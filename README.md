📂 Projetos Desenvolvidos
1. [Mini Sistema de Agendamento Esportivo](https://github.com/lucas253503/sistema-agendamento-esportivo)
.Descrição: Uma solução web completa voltada para o gerenciamento de reservas de horários em quadras e ambientes de lazer (baseado no sistema Arena Lazer). 
.O que faz: Permite que o usuário selecione a modalidade esportiva desejada, escolha a data e o horário disponível, preencha os dados e realize o agendamento.
.Destaque Técnico: Os dados do agendamento são persistidos localmente por meio do navegador, e o JavaScript valida os campos para evitar conflitos de horários repetidos.

2. [Calculadora Simples](https://github.com/lucas253503/calculadora-simples)
.Descrição: Uma aplicação de calculadora digital clássica feita para rodar diretamente no navegador do usuário.
.O que faz: Realiza todas as operações matemáticas fundamentais (soma, subtração, multiplicação e divisão) com uma interface interativa que simula um dispositivo real.
.Destaque Técnico: Focado no estudo de escuta de eventos (*event listeners*) do JavaScript, captura de strings no visor e processamento matemático de dados em tempo real.
